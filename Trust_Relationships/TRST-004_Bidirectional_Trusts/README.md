# Bidirectional Trusts

## Description
Lists trusts that are bidirectional (trustDirection=3). Higher risk.

## Severity
HIGH

## Category
Trust Relationships

## Remediation
Minimize bidirectional trusts. Consider one-way trusts where possible.
