# Computer Accounts with Old Password (1+ Year)

## Description
Finds computers that have not changed password in over 1 year.

## Severity
MEDIUM

## Category
Computers & Servers

## Remediation
Investigate stale machine passwords. May indicate offline or orphaned accounts.
