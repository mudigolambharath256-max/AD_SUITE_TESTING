# Computers with Pre-2000 Compatible Access

## Description
Checks for computers in Pre-Windows 2000 Compatible Access group.

## Severity
MEDIUM

## Category
Computers & Servers

## Remediation
Remove from Pre-Windows 2000 Compatible Access unless specifically required.
