# Computers Running Unsupported OS

## Description
Finds computers running Windows XP, Vista, 7, Server 2003, 2008, or 2012.

## Severity
CRITICAL

## Category
Computers & Servers

## Remediation
Upgrade or decommission unsupported systems. If needed, isolate on separate network segment.
