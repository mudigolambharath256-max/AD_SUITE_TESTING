# Computers with S4U Delegation

## Description
Finds computers with TRUSTED_TO_AUTH_FOR_DELEGATION (protocol transition).

## Severity
HIGH

## Category
Computers & Servers

## Remediation
Review S4U configurations. Ensure least privilege on delegation targets.
