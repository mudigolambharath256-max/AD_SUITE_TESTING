# Computers with KeyCredentialLink

## Description
Finds computers with msDS-KeyCredentialLink (shadow credentials).

## Severity
MEDIUM

## Category
Computers & Servers

## Remediation
Verify all KeyCredentialLink entries are from legitimate device registration.
