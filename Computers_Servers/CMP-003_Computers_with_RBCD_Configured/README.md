# Computers with RBCD Configured

## Description
Lists computers with Resource-Based Constrained Delegation (msDS-AllowedToActOnBehalfOfOtherIdentity).

## Severity
HIGH

## Category
Computers & Servers

## Remediation
Audit RBCD configurations. Ensure only authorized principals are listed.
