# Stale Computer Accounts (90+ Days)

## Description
Finds computer accounts that have not authenticated in 90+ days.

## Severity
MEDIUM

## Category
Computers & Servers

## Remediation
Disable stale accounts, quarantine for 30 days, then delete.
