# FRS SYSVOL (Deprecated)

## Description
Checks if FRS is still used for SYSVOL (deprecated).

## Severity
HIGH

## Category
Domain Controllers

## Remediation
Migrate SYSVOL replication from FRS to DFSR immediately.
