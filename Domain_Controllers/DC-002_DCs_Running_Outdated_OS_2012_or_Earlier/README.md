# DCs Running Outdated OS (2012 or Earlier)

## Description
Finds Domain Controllers running Windows Server 2012 R2 or earlier (out of support).

## Severity
CRITICAL

## Category
Domain Controllers

## Remediation
Upgrade all DCs to Windows Server 2019 or later. Create migration plan with proper backup.
