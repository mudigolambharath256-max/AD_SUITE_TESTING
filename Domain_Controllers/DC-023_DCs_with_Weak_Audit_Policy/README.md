# DCs with Weak Audit Policy

## Description
Checks if DCs have insufficient audit logging enabled for security events.

## Severity
HIGH

## Category
Domain Controllers

## Remediation
Enable comprehensive audit policy via Group Policy. Audit logon events, account management, policy changes.
