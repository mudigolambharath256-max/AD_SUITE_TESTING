# DCs with Anonymous SID Translation Enabled

## Description
Checks if anonymous SID/Name translation is allowed. This enables information disclosure.

## Severity
HIGH

## Category
Domain Controllers

## Remediation
Disable anonymous SID translation via Group Policy or registry.
