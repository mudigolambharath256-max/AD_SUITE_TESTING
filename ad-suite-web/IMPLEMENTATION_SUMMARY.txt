KEY FEATURES
================================================================================

Context Injection:
  - $global:domain = 'corp.domain.local'
  - $global:domainDN = 'DC=corp,DC=domain,DC=local'
  - $global:targetServer = '192.168.1.10'

Quick Commands (Dynamic):
  - Always: whoami, hostname, ipconfig, $PSVersionTable
  - With IP: ping, LDAP :389, LDAPS :636, GC :3268, Kerberos :88, RootDSE

  - Minimized (44px) - Header only, session alive
  - Normal (380px) - Full terminal
  - Expanded (620px) - More viewing space

Session Management:

  - 30-minute idle timeout

  - Session logging (timestamps only)

================================================================================
TROUBLESHOOTING
================================================================================

If terminal won't open:
  - Check backend console for "[Terminal] WebSocket server attached"
  - Check browser console (F12) for errors
  - Verify WebSocket proxy in vite.config.js

If PowerShell not found:
  - Ensure powershell.exe is in system PATH
  - Should be default on Windows

If connection lost:
  - Click "Reconnect" button
  - Check backend server is running
