# GPO Containing Password (Naming)

## Description
Finds GPOs with "password" in the name (may contain GPP creds).

## Severity
HIGH

## Category
Group Policy

## Remediation
Review for Group Policy Preferences passwords. These are easily decrypted (MS14-025).
