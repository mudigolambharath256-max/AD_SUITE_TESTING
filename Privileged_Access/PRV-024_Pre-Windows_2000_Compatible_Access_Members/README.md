# Pre-Windows 2000 Compatible Access Members

## Description
Lists members of Pre-Windows 2000 Compatible Access (exposes anonymous access).

## Severity
HIGH

## Category
Privileged Access

## Remediation
Remove Anonymous Logon and Everyone from this group.
