# Admin Accounts That Can Be Delegated

## Description
Finds admin accounts without "Cannot be delegated" flag.

## Severity
HIGH

## Category
Privileged Access

## Remediation
Enable "Account is sensitive and cannot be delegated" for admin accounts.
