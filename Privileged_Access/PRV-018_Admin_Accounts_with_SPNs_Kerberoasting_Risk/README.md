# Admin Accounts with SPNs (Kerberoasting Risk)

## Description
Finds privileged accounts with SPNs set.

## Severity
CRITICAL

## Category
Privileged Access

## Remediation
Remove SPNs from admin accounts. Migrate services to gMSA.
