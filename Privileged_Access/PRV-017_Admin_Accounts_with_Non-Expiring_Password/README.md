# Admin Accounts with Non-Expiring Password

## Description
Finds privileged accounts with DONT_EXPIRE_PASSWORD set.

## Severity
CRITICAL

## Category
Privileged Access

## Remediation
Enable password expiration for admin accounts or use gMSA.
