# Membership: Enterprise Admins

## Description
Lists members of the privileged group "Enterprise Admins".

## Severity
CRITICAL

## Category
Privileged Access

## Remediation
Minimize Enterprise Admins membership. Use tiered admin model, JIT access, and dedicated admin accounts.
