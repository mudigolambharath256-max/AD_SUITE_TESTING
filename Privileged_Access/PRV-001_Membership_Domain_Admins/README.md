# Membership: Domain Admins

## Description
Lists members of the privileged group "Domain Admins".

## Severity
CRITICAL

## Category
Privileged Access

## Remediation
Minimize Domain Admins membership. Use tiered admin model, JIT access, and dedicated admin accounts.
