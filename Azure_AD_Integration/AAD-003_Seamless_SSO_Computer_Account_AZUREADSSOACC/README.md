# Seamless SSO Computer Account (AZUREADSSOACC)

## Description
Finds the Seamless SSO computer account used for Kerberos authentication.

## Severity
HIGH

## Category
Azure AD Integration

## Remediation
Rotate AZUREADSSOACC password every 30 days. Protect from DCSync.
