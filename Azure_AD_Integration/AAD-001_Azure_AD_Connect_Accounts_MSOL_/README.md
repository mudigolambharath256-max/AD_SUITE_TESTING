# Azure AD Connect Accounts (MSOL_*)

## Description
Lists Azure AD Connect sync accounts (MSOL_ prefix).

## Severity
HIGH

## Category
Azure AD Integration

## Remediation
MSOL accounts are highly privileged. Monitor closely and rotate credentials per vendor guidance.
