# Accounts with ms-DS-ConsistencyGuid

## Description
Lists accounts with Azure AD sync anchor attribute.

## Severity
INFO

## Category
Azure AD Integration

## Remediation
ConsistencyGuid is the sync anchor. Do not modify manually.
