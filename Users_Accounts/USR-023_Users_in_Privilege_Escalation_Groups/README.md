# Users in Privilege Escalation Groups

## Description
Lists users in groups that enable privilege escalation (Backup Operators, Account Operators, etc.).

## Severity
HIGH

## Category
Users & Accounts

## Remediation
Minimize membership in operator groups. These groups have paths to Domain Admin.
