# Locked Accounts

## Description
Finds currently locked out accounts. May indicate brute-force or password spray attack.

## Severity
MEDIUM

## Category
Users & Accounts

## Remediation
Investigate lockouts. Check security logs for password spray attacks (Event ID 4740).
