# Accounts with unicodePwd Readable

## Description
Lists accounts where unicodePwd might be exposed (should never be readable).

## Severity
INFO

## Category
Users & Accounts

## Remediation
unicodePwd should never be readable. If you can read it, there is a serious misconfiguration.
