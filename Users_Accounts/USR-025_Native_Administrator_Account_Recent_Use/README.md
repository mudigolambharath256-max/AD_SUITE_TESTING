# Native Administrator Account Recent Use

## Description
Checks if built-in Administrator account (RID 500) has been used recently.

## Severity
MEDIUM

## Category
Users & Accounts

## Remediation
Disable built-in Administrator or rename and use only for emergency. Create named admin accounts.
