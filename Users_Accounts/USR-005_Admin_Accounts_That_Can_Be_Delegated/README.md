# Admin Accounts That Can Be Delegated

## Description
Finds admin accounts without "Account is sensitive and cannot be delegated" flag.

## Severity
HIGH

## Category
Users & Accounts

## Remediation
Enable "Account is sensitive and cannot be delegated" for all admin accounts.
