# Inactive Accounts (90+ Days)

## Description
Finds accounts that have not logged on in 90+ days.

## Severity
MEDIUM

## Category
Users & Accounts

## Remediation
Disable accounts inactive for 90+ days. Delete after 180 days of inactivity.
