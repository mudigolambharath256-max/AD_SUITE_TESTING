# Accounts with altSecurityIdentities

## Description
Finds accounts with altSecurityIdentities attribute set. Can be used for privilege escalation via certificate mapping.

## Severity
MEDIUM

## Category
Users & Accounts

## Remediation
Review and remove unauthorized altSecurityIdentities mappings. Restrict who can modify this attribute.
