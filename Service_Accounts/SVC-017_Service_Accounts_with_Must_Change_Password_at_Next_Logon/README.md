# Service Accounts with “Must Change Password at Next Logon”

## Description
Lists SPN accounts with pwdLastSet=0 (operational issue / risk).

## Severity
MEDIUM

## Category
Service Accounts

## Remediation
Set a strong long random password and rotate securely, or migrate to gMSA.
