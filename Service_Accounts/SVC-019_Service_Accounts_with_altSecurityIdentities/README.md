# Service Accounts with altSecurityIdentities

## Description
Lists SPN accounts with altSecurityIdentities mappings.

## Severity
MEDIUM

## Category
Service Accounts

## Remediation
Review and restrict altSecurityIdentities. Remove unused mappings.
