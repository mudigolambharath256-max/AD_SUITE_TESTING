# Service Accounts with LogonWorkstations Restriction

## Description
Lists SPN accounts with userWorkstations set (inventory for restrictions).

## Severity
INFO

## Category
Service Accounts

## Remediation
Ensure service accounts are non-interactive and restricted by policy (deny interactive logon).
