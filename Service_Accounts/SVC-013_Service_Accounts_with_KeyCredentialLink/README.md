# Service Accounts with KeyCredentialLink

## Description
Lists SPN accounts with msDS-KeyCredentialLink populated (review for unexpected keys).

## Severity
HIGH

## Category
Service Accounts

## Remediation
Validate any device/WHfB keys. Remove unknown keys and investigate.
