# Service Accounts with Password Never Expires

## Description
Finds service accounts with DONT_EXPIRE_PASSWORD set.

## Severity
CRITICAL

## Category
Service Accounts

## Remediation
Enforce rotation for service account passwords or migrate to gMSA.
