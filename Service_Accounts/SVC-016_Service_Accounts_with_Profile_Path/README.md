# Service Accounts with Profile Path

## Description
Lists service accounts with roaming profile configured (should be rare).

## Severity
MEDIUM

## Category
Service Accounts

## Remediation
Remove interactive configuration from service accounts and ensure “Deny interactive logon” policies.
