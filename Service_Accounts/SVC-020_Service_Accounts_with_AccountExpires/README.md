# Service Accounts with AccountExpires

## Description
Lists SPN accounts with accountExpires set (inventory for lifecycle management).

## Severity
INFO

## Category
Service Accounts

## Remediation
Ensure service accounts are lifecycle-managed and decommissioned when services retire.
