# Service Accounts with UPN Suffix Outside Standard

## Description
Lists SPN accounts with UPN set (review identity governance).

## Severity
INFO

## Category
Service Accounts

## Remediation
Ensure UPNs align with governance and are not used to enable unintended cloud auth paths.
