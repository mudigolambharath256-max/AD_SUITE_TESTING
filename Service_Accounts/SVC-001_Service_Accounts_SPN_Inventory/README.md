# Service Accounts (SPN Inventory)

## Description
Lists enabled user accounts with SPNs (service accounts).

## Severity
INFO

## Category
Service Accounts

## Remediation
Ensure each SPN maps to a known service. Remove obsolete SPNs and decommission unused services.
