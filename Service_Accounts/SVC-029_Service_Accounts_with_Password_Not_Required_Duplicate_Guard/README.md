# Service Accounts with “Password Not Required” (Duplicate Guard)

## Description
Secondary listing of PASSWD_NOTREQD on SPN accounts.

## Severity
CRITICAL

## Category
Service Accounts

## Remediation
Unset PASSWD_NOTREQD and rotate the credential immediately.
