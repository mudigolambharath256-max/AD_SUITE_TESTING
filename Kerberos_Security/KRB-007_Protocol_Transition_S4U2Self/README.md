# Protocol Transition (S4U2Self)

## Description
Finds accounts with TRUSTED_TO_AUTH_FOR_DELEGATION (S4U protocol transition).

## Severity
HIGH

## Category
Kerberos Security

## Remediation
S4U2Self allows impersonating any user. Review and restrict delegation targets.
