# AS-REP Roastable + SPN (Double Risk)

## Description
Finds accounts vulnerable to both AS-REP roasting and Kerberoasting.

## Severity
CRITICAL

## Category
Kerberos Security

## Remediation
Highest priority: Enable pre-auth and migrate to gMSA.
