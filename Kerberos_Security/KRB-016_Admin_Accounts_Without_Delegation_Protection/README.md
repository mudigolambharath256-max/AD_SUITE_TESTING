# Admin Accounts Without Delegation Protection

## Description
Finds admin accounts missing "Account is sensitive and cannot be delegated" flag.

## Severity
HIGH

## Category
Kerberos Security

## Remediation
Enable "Account is sensitive and cannot be delegated" for all admin accounts.
