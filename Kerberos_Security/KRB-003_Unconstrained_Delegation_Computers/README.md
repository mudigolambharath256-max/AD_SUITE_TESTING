# Unconstrained Delegation (Computers)

## Description
Finds computers trusted for delegation (excluding DCs). Critical attack vector.

## Severity
CRITICAL

## Category
Kerberos Security

## Remediation
Remove unconstrained delegation. Use constrained delegation or RBCD instead.
