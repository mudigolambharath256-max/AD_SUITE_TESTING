# ESC2: Templates with Any Purpose EKU

## Description
Finds templates with Any Purpose or no EKU (can be used for anything including auth).

## Severity
CRITICAL

## Category
Certificate Services

## Remediation
Remove Any Purpose EKU and specify explicit usage. Restrict enrollment rights.
