# ESC7: CA with Manage Certificates Permission

## Description
Lists CAs for ACL review (manual check for Manage Certificates permission).

## Severity
HIGH

## Category
Certificate Services

## Remediation
Restrict Manage Certificates permission to CA administrators only.
