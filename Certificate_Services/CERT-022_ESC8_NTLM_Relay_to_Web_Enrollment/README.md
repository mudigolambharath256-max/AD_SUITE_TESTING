# ESC8: NTLM Relay to Web Enrollment

## Description
Lists CAs that may have web enrollment enabled (requires network verification).

## Severity
HIGH

## Category
Certificate Services

## Remediation
Disable NTLM on CA web enrollment. Enable EPA and require HTTPS.
