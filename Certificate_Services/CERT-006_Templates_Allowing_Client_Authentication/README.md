# Templates Allowing Client Authentication

## Description
Finds templates with Client Authentication EKU that enable domain authentication.

## Severity
HIGH

## Category
Certificate Services

## Remediation
Restrict enrollment. Ensure SAN cannot be specified by enrollee.
