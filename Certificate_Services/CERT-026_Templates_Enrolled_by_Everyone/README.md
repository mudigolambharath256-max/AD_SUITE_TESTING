# Templates Enrolled by Everyone

## Description
Lists templates to check for overly permissive enrollment (manual ACL review).

## Severity
HIGH

## Category
Certificate Services

## Remediation
Restrict enrollment to specific security groups. Remove Everyone/Authenticated Users.
