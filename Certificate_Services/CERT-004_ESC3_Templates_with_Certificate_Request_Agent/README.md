# ESC3: Templates with Certificate Request Agent

## Description
Finds templates with Certificate Request Agent EKU (can enroll on behalf of others).

## Severity
CRITICAL

## Category
Certificate Services

## Remediation
Restrict enrollment to highly trusted accounts. Require manager approval.
