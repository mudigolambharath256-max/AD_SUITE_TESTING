🎯 How to Access the Attack Path Graphical Viewer

1

Open the AD Security Suite

Go to: http://localhost:5173

2

Navigate to Attack Path Analysis

Click on "Attack Path Analysis" in the sidebar navigation

✅ This page shows the graphical attack path viewer

3

Choose Data Source

Select "Upload JSON/CSV" option (this is the easiest way to test)

⚠️ Don't select "Most recent scan" - it requires actual scan results

4

Load Sample Data (EASIEST METHOD)

Click one of the sample buttons:

  • 🎭 Load GOAD Sample - Game of Thrones themed attack scenario
  • 🏢 Load Advanced Sample - Enterprise attack scenario

✅ This instantly loads realistic attack findings!

5

Configure OpenAI Settings

  • Provider: Select OpenAI
  • Model: Select GPT-4o Mini (recommended)
  • API Key: Enter your OpenAI API key

✅ Your key works with GPT-4o-mini!

6

Run the Analysis

Click the "Analyze" button

✅ This generates the attack path graph and narrative!

7

View the Graphical Attack Path

You will see:

  • 🎨 Interactive Graph with nodes and edges
  • 🔴 Color-coded Severity (CRITICAL=red, HIGH=orange, etc.)
  • 📊 Attack Relationships showing privilege escalation paths
  • 📝 Professional Narrative explaining attack chains
  • 🎯 MITRE ATT&CK Mapping for each technique

🎯 Alternative: Upload Your Own Files

If you want to use the test files I created:

  1. Download: sample_goad_findings.json
  2. Or: advanced_attack_scenario.json
  3. Click the upload zone and select the file
  4. Proceed with steps 5-6 above

🌟 What You'll See

✅ Interactive Attack Graph - Drag nodes, zoom, pan

✅ Severity-Based Colors - Risk level visualization

✅ Attack Paths - How attackers escalate privileges

✅ Security Narrative - Professional analysis report

✅ MITRE Techniques - ATT&CK framework mapping

🚀 Quick Start

Fastest way to see the graphical viewer:

  1. Open http://localhost:5173
  2. Click "Attack Path Analysis"
  3. Select "Upload JSON/CSV"
  4. Click "🎭 Load GOAD Sample"
  5. Select "OpenAI" → "GPT-4o Mini"
  6. Enter your API key
  7. Click "Analyze"
